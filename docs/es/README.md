# PatchQuest: un taller de agentes

[English](../en/README.md) · Español

PatchQuest es una referencia tecnológica neutral para aprender diseño dirigido
por el dominio, arquitectura hexagonal y microservicios fiables mediante un
flujo de trabajo con agentes.

Una persona abre una misión de programación. Un ejecutor obtiene temporalmente
un intento y entrega un artefacto. Un verificador independiente comprueba ese
artefacto exacto. Finalmente, una persona decide si el resultado completa la
misión.

La documentación inglesa es la fuente normativa. La traducción completa al
español se realizará y revisará por personas cuando la documentación inglesa se
estabilice. Los identificadores técnicos, nombres de eventos, claves de esquemas,
comandos y diagramas no se traducirán.

Por ahora, consulta la [guía normativa en inglés](../en/README.md) y el
[README principal](../../README.md).
